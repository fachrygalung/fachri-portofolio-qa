# Robot Framework - Sauce Demo Automation

Automation testing menggunakan Robot Framework dengan SeleniumLibrary pada website Sauce Demo.

## Platform

- Websites

## Tech Stack

- Robot Framework 7.4.2
- SeleniumLibrary 6.9.0
- Python 3.10
- Chrome Browser
- VS Code Editor

## Run Test

Run all test cases:

```bash
robot tests/
```

Run login test only:

```bash
robot tests/login.robot
```
